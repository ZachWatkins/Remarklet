# Remarklet

Remarklet adds visual editing capabilities to any web page. This makes modifying and commenting on web pages remarkably easy!

# Features

-   Drag elements around the page
-   Resize elements
-   Edit text

# Feature Roadmap

-   Insert Note
-   Delete Elements
-   View Grid
-   Copy Elements
-   Export the edited page as an HTML file
-   Insert Image
-   Insert Code
-   Save
-   Restore
